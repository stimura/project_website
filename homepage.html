Summary:

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this task, I first pulled data from the OpenWeatherMap API to assemble a dataset on 500 cities worldwide. Using Matplotlib, I constructed multiple plots depicting the relationship between weather and latitude. The factors I selected to demonstrate the relationsip between weather and latitude included: temperature, cloudiness, wind speed, and humdity. I constructed this site for users to easily examine and digest the somewhat suprising nature of cities' latitudes with respect to certain weather conditions. Please use the dropdown menu on the navbar to view detailed analysis of the various weather conditions that I examined. Additionally, I incorporated a webpage displaying the data I used for this project.
